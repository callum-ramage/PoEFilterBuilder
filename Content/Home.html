Building a filter is done on the filter tab by pressing the compile button.
This button saves and builds the selected filter from the text area and uses the name in the name field.

The process of building a filter replaces tags inside {} with the content of the replacement with that name.
This is a recursive process where replacements can also have replacements. I'm not protecting against recursion depth so if it falls over, that's on you.

Replacements

Replace Format: {replace file name,option 1,...,option N}
{currency,5}
{talisman}

Parameter Format: [n] (1 indexed)
[1]
